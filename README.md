

## Installation

### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/Coopathon2019/coopathon-web.git Coopathon-Web

# go into app's directory
$ cd Coopathon-Web

# install app's dependencies
$ npm install
```

## Usage

``` bash
# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build
